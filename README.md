# Star Wars Character/Movie Guide  
🔭 I’m currently working on the **Star Wars Character/Movie Guide** assignment. This project involves researching the Star Wars API and creating a responsive guide. The guide will display a list of characters and, upon interaction, fetch movie details using AJAX, including the movie title, opening crawl, and poster.

## Assignment Description

Working in pairs, you will research the [Star Wars API](https://swapi.dev) and create a character/movie guide. The guide should include an unordered list with 10+ characters, each linking to a movie they appeared in. When a character is clicked, an AJAX call will fetch and display the movie title, opening crawl, and poster image.  

The guide must be responsive from mobile to desktop. GreenSock enhancements are encouraged, and you must use HTML templates or template literals for rendering content. Make sure your Fetch functions handle different AJAX request stages and failures, and implement a loading icon to indicate content is loading.
